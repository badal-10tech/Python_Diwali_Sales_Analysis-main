# 🪔 Diwali Sales Data Analysis (EDA) using Python

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a **Diwali Sales dataset** using **Python**. The objective of the analysis is to uncover useful insights about customer purchasing behavior during the Diwali festival season.

Diwali is one of the biggest shopping festivals in India, and analyzing sales data can help businesses understand:

* Customer demographics
* Buying patterns
* Product demand
* Revenue distribution

The project uses Python libraries to clean, analyze, and visualize the data to extract meaningful business insights.

---

## 📂 Dataset

The dataset contains transactional sales data during the **Diwali festival period**, including customer demographics and purchase information.

Typical columns in the dataset include:

* `User_ID`
* `Cust_name`
* `Product_ID`
* `Gender`
* `Age Group`
* `Age`
* `Marital_Status`
* `State`
* `Occupation`
* `Product_Category`
* `Orders`
* `Amount`

The dataset helps analyze **which customers are purchasing more and which products are performing best during Diwali sales.**

## 📊 Project Workflow

### 1️⃣ Data Collection

The Diwali sales dataset was loaded into a **Pandas DataFrame** for analysis.

### 2️⃣ Data Cleaning

Data preprocessing steps included:

* Removing null values
* Dropping unnecessary columns
* Fixing data types
* Handling missing or inconsistent values

### 3️⃣ Exploratory Data Analysis (EDA)

EDA was performed to understand the dataset and discover trends using visualizations.

Key analysis includes:

#### Gender Analysis

* Comparison of purchases made by **male vs female customers**

#### Age Group Analysis

* Identifying which **age groups contribute most to sales**

#### State-wise Sales

* Understanding which **states generate the highest revenue**

#### Marital Status Analysis

* Purchase behavior based on **marital status**

#### Occupation Analysis

* Determining which **occupations spend the most during Diwali**

#### Product Category Analysis

* Identifying the **most popular product categories**

---

## 📈 Key Insights from the Analysis

Some of the major insights discovered during the analysis include:

* **Married women** in the **age group of 26–35 years** are the highest buyers during the Diwali season.
* Customers from states like **Uttar Pradesh, Maharashtra, and Karnataka** contribute significantly to total sales.
* People working in **IT, Healthcare, and Aviation sectors** tend to spend more during the festive season.
* **Food, Clothing, and Electronics** are among the most popular product categories.
